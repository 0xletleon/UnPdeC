# UnPdeC

    使用 腾讯元宝，Kimi 辅助将C#转换为C++代码
    算是学习C++代码了，请勿用于任何商业目的！


## Build

    C++17 标准

    项目属性 -> 配置属性 -> 常规 -> C++ 语言标准

    改成 ISO C++17 标准 (/std:c++17)
